# nv-atlas ロードマップ

## 完了済み

### キーボードビジュアル + Vim操作表示
- VIA 定義 JSON（KLE形式）を読み込んでキーボードを描画
- VIA キーマップ JSON を読み込んでベースレイヤーのキー割り当てを反映
- 各キーに出力文字と対応する Vim コマンドを表示
- キーをホバーすると Vim コマンドの詳細説明を表示
- Vim コマンドのカテゴリ別色分け（移動、編集、検索など）
- スプリットキーボード（Corne 等）のカラムスタガー・回転サムクラスターに対応
- レイアウトオプション（encoder/extra key）のフィルタリング

### インタラクティブ練習モード
- 「`j` を押して下に移動」のようなプロンプトを表示
- ユーザーが正しい物理キーを押せたかを判定
- キーボード上で押すべきキーをハイライト
- 正答率の記録
- カテゴリ別の練習（移動だけ、編集だけ、等）

### 辞書モード（CommandReference）
- Vim コマンドの一覧検索・参照

### 操作単位キーバインド編集 — データレイヤー
- `Keybinding` / `KeybindingConfig` 型定義
- vimCommands からデフォルト設定を生成
- カスタム配列からプリセット生成（`deriveFromLayout`）
- `useReducer` ベースの状態管理 + Context

### Vim モードセレクタ
- VimCommand に `modes` フィールド追加
- モード切替タブ（ModeSelector）
- モード別キーバインド表示

## キーバインド編集の充実

### 編集 UI
- BindingEditor コンポーネント（操作ごとにキーを再割当）
- KeyCapture（キー入力キャプチャ）

### Neovim インポート強化
- 全モード対応のキーマップインポート

### Lua/JSON エクスポート
- 現在のキーバインド設定を Neovim 用 Lua や JSON で出力

### localStorage 永続化 + プリセット
- 設定の保存・復元
- プリセット配列（Colemak, Dvorak 等）の同梱

### Practice/Reference モード適応
- 練習モード・辞書モードがカスタムキーバインドを反映
