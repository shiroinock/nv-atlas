# KeyViz ロードマップ

## Phase 1: キーボードビジュアル + Vim操作表示 [完了]

- VIA 定義 JSON（KLE形式）を読み込んでキーボードを描画
- VIA キーマップ JSON を読み込んでベースレイヤーのキー割り当てを反映
- 各キーに出力文字と対応する Vim コマンドを表示
- キーをホバーすると Vim コマンドの詳細説明を表示
- Vim コマンドのカテゴリ別色分け（移動、編集、検索など）
- スプリットキーボード（Corne 等）のカラムスタガー・回転サムクラスターに対応
- レイアウトオプション（encoder/extra key）のフィルタリング

## Phase 2: インタラクティブ練習モード

- 「`j` を押して下に移動」のようなプロンプトを表示
- ユーザーが正しい物理キーを押せたかを判定
- キーボード上で押すべきキーをハイライト
- 正答率の記録
- カテゴリ別の練習（移動だけ、編集だけ、等）

## Phase 3: チュートリアル読み替え

- Vim チュートリアルのテキストを貼り付けると、キーバインドを自分の配列に読み替えた版を生成
- 「`dd` で行削除」→「`rr` で行削除（あなたの配列）」のような変換
- コンビネーションキー（`dw`, `ci"` 等）にも対応
